# STUDENT MANAGEMENT SYSTEM (Python + MySQL)

## Description

This project is built using Python and MySQL.
This is a console-based application that performs CRUD operations.
It helps to manage student records like adding, viewing, updating, and deleting students.

## Features
- Add Student
- View Student
- Update Student
- Delete Student

## Technologies Used
- Python
- MySQL
- mysql-connector

## How to Run
1. Install mysql.connector:
    pip install mysql-connector-python
2. Setup MySQL Database
3. Run the Python File
