# 📚 Online Shopping Cart (Python + MySQL)

## 📌 Description
ShopKit is a console-based shopping cart application built using Python and MySQL.
It allows users to view products, add items to a cart, remove items, and calculate the total price.
The project demonstrates basic database connectivity and CRUD operations using Python.

## 🚀 Features
- View Available Products
- Add Products to Cart
- View Cart Items
- Remove Products from Cart
- Automatic Total Price Calculation

## 🛠 Technologies Used
- Python
- MySQL
- PyMySQL (Database Connector)

## How to Run

- Install PyMySQL:
    pip install pymysql
    Make sure MySQL is running on your system.

- Update your MySQL credentials in the code:
    host = 'localhost'
    user = 'root'
    password = 'your_password'

- Run the Python script:
    python your_file_name.py
