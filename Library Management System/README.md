# 📚 Library Management System (Python + MySQL)

## 📌 Description
This project is built using Python and MySQL.
It is a console-based application that manages library books.
It performs CRUD operations and allows issuing and returning books.

## 🚀 Features
- Add Book
- View Books
- Issue Book
- Return Book
- Delete Book

## 🛠 Technologies Used
- Python
- MySQL
- mysql-connector

## ▶️ How to Run
1. Install mysql-connector:
   pip install mysql-connector-python
2. Setup MySQL Database
3. Run the Python file
